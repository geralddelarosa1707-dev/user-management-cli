# User Account Management CLI App

A Python CLI app that handles user account management, including registration, login, validation, and JSON data persistence.

## Features
- Create account with validation
- Login system with attempt limits
- Change username and password
- Logout account
- Delete account
- JSON-based data persistence
- Structured logging for debugging

## Tech Used
- Python
- JSON (file storage)
- hashlib (SHA-256 for password hashing)

## Notes
- Password hashing uses SHA-256 without salting (for learning purposes only)
- Email validation is basic and not production-ready
